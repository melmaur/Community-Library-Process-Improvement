# AS-IS Process

## Current Workflow

1. Member searches for a book.
2. Library catalogue is checked.
3. If available, a reservation request is submitted.
4. Reservation is recorded in the booking system.
5. Member receives confirmation.
6. Book is collected.
7. Book is returned.

## Stakeholders

- Library Member
- Librarian
- Library System

## Observed Limitations

- No waiting list process
- No automated reminders
- Limited operational reporting
- No overdue notification workflow
