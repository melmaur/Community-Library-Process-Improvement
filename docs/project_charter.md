# Project Charter

## Project Title

Community Library Process Improvement Initiative

---

## Purpose

The purpose of this initiative is to analyse the current library borrowing and reservation process, 
identify operational inefficiencies, and propose improvements to enhance user experience and service delivery.

---

## Background

The Community Library Platform currently consists of three services developed in Java:

- Library Management
- Library REST API
- Booking System

While the technical solution supports library operations, a formal review of the end-to-end business process has not been conducted.

---

## Objectives

- Document the current borrowing process
- Identify bottlenecks and inefficiencies
- Define an improved future-state process
- Produce business requirements
- Recommend operational improvements

---

## Deliverables

- Project Charter
- AS-IS Process Map
- TO-BE Process Map
- Gap Analysis
- Requirements Document
- Improvement Recommendations

---

## Success Criteria

- Current process documented
- Improvement opportunities identified
- Requirements defined
- Recommendations approved
